### config example:

```json
{
  "Port": 8080,
  "DbUsername": "admin",
  "DbPassword": "root",
  "DbUrl": "localhost:27017",
  "DbName": "YOUR_DB_NAME",
  "Secret": "YOUR_SECRET",
  "SPARKPOST_API_KEY": "YOUR_API_KEY"
}
```

##### Please change the parameters to suit with local dev env and then put the config.json into config dir.
